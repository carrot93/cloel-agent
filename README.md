# cloel-agent

emacs ai agent based on E2EE(Emacs 2 platform Enterprise Edition) E/S structure.

## How to 

```
cd ~/.emacs.d
git clone https://github.com/carrot93/cloel-agent/
cd cloel-agent
clojure -P
```

## In Emacs

M-x cloel-agent-run

## agent and skills directory

```
~/.emacs.d/cloel-agent/agent
```

## support platform

Windows/Linux/FreeBSD/MacOS
